1. Запустить OS
2. Переключиться в консольный режим Alt+Ctrl+F2
3. Авторизоваться целевым пользователем в консольном режиме.
4. Выполнить команду "startx -- :1 ; logout"
5. Переключиться на запущенный Xserver -> Alt+Ctrl+F8 (переключение осуществляется автоматически)
6. Для переключения на дефолтный Xserver -> Alt+Ctrl+F7

Подобным образом можно запустит 4ре сессии.
1.
	Alt+Ctrl+F2
	"startx -- :1 ; logout"
	Alt+Ctrl+F8

2.
	Alt+Ctrl+F3
	"startx -- :2 ; logout"
	Alt+Ctrl+F9
