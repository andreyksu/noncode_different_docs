java.lang.reflect.Proxy

Конструирует объект во время исполнения программы (динамически) - при этом не нужно создавать для него отдельного класса.

В Java используется специальный интерфейс InvocationHandler, с помощью которого можно перехватывать все вызовы методов, обращенные к proxy-объекту. 
proxy-объект можно создать только используя интерфейсы.

У интерфейса InvocationHandler есть единственный метод invoke, в который направляются все вызовы, обращенные к proxy-объекту.
